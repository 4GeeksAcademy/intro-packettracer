Configuring DHCP

This protocol is responsible for dynamically assigning IP addresses to all devices connected to the network. To do this, we must create a pool of addresses from which to assign to the devices that request them. In addition to the IP address, the gateway and subnet mask are also provided.

On "Server 0"

  1. In the server configuration, select the "Services" tab.
  2. In the left panel, select the "DHCP" service. This will display all the configuration options for this service.
  3. First, activate the service and verify that the interface you configured in the previous step is selected: "Ethernet0".

Configuración de DHCP

Este protocolo se va a encargar de asignar direcciones IP de manera dinámica a todos los dispositivos conectados a la red. Para ello debemos crear una piscina (pool) de direcciones de las cuales se tomarán para asignar a los equipos que las vayan solicitando. Ademas de la dirección IP también se proporciona la puerta de enlace y la mascara de subred.

En el "Server 0"

  1. Dentro de la configuración del servidor selecciona la pestaña "Services"
  2. En el panel izquierdo selecciona el servicio "DHCP", ahi verás como se despliegan todas las opciones de configuración de este servicio
  3. Primero activa el servicio y verifica que este seleccionada la interfaz que configuraste en el paso previo: "Ethernet0"
  1. Next, we need to configure the following parameters for our IP address pool:
    • Default Gateway: 192.168.0.1
    • Start IP Address: 192.168.0.100
    • Subnet Mask: 255.255.255.0
    • DNS server IP: 8.8.8.8
    • Maximum Number of Users: 100
    Once everything is configured, we must click the save button to update the pool.
